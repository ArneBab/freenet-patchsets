About this page
This page is an effort to start decentralizing freenet development and push it into freenet itself.

To achieve that goal, it is writable for longterm developers.

All its code is available from an infocalypse repository within Freenet (use hg fn-clone COPIED_URI; whatâ€™s infocalypse? - non-anonymous version), from bitbucket and from github.
